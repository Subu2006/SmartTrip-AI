## ✨ About SmartTrip AI

SmartTrip AI is a next-generation intelligent travel platform designed to solve fragmented travel planning.

Instead of using 5–7 apps for booking, planning, budgeting, maps, and finding companions, SmartTrip AI unifies everything into one powerful platform.

Whether you're a solo traveler, student, family planner, backpacker, or adventure seeker — SmartTrip AI gives you:

✅ Personalized destination recommendations  
✅ AI-generated trip itineraries  
✅ Smart budget optimization  
✅ Travel buddy matching  
✅ Expense tracking  
✅ Live chat  
✅ Booking integrations  
✅ Maps + Weather insights  
✅ Admin analytics dashboard

## ⚙️ Installation

```bash
git clone https://github.com/yourusername/SmartTrip-AI.git

cd SmartTrip-AI

npm install

cp .env.example .env

npm run dev
```
